まず、最初にソースをダウンロードしてくれてありがとうございました。
バグの箇所を教えて頂いたりすると、作者はかなりやる気が出ます。
機能追加をしてくれると、飛び跳ねると思います。

そんな方のために、設計の外観を説明したいと思います。


最初の意図は
・Controller が全てを取り仕切って
・*Wnd は Controller から表示しろと言われた内容だけを表示する
という感じでした。ところが、*Wnd が出しゃばってきて意味不明になってます。
特に MainWnd と ListWnd の関係が複雑で分かりにくいかと思います。

〜の動作をする、と言うときに Controller と MainWnd の線引きも微妙です。

読みにくくてすいません。
今思うと、ListWnd が ZipFile の内容を表示する Doc-View モデルぐらいにして
おいた方がよかった気がします。


ついでに言うと、*File は 勢いで書いたから意味不明です。
